Array对象内置方法总结

发表于 2017-06-23

1. Array.from()

作用：从一个类似数组或可迭代的对象中创建一个新的数组实例
语法：Array.from(arraylike, mapFn, thisArg);
参数：

  • arraylike: 类数组对象或可遍历对象(有length属性的对象)
  • mapFn: 可选参数，用来处理数组的函数，返回新数组
  • thisArg: 可选参数，执行mapFn时的this值

返回值：新的Array实例
是否改变原数组的值：false
兼容性：IE-
版本：ES5
描述：Array.from(obj, mapFn, thisArg) 类似于 Array.from(obj).map(mapFn, thisArg)

const bar = ['a','b','c'];
var newArr = Array.from(bar);//['a','b','c']

//等同于string.split("");
var newArr = Array.from('foo') //['f','o','o'];

var newArr = Array.from({length: 5},function(val,i){
    return i;
}); //[0,1,2,3,4];

2. Array.isArray()

作用：用于确定传递的值是否是一个 Array
语法：Array.isArray(obj)
参数：

  • obj: 需要检测的值

返回值：array ? true : false
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：—

Array.isArray([1, 2, 3]);  
// true
Array.isArray({foo: 123}); 
// false

3. Array.of()

作用：创建一个具有可变数量参数的新数组实例,而不考虑参数的数量或类型
语法：Array.of(elem0, elem1, elem2, …elemN)
参数：任意个参数，将按顺序成为返回数组中的元素
返回值：新的 Array 实例
是否改变原数组的值：false
兼容性：IE-,Opera-
版本：ES6
描述：—

var newArr = Array.of(1) //[1]
var newArr = Array.of(1,2,3) //[1,2,3]
var newArr = Array.of("zhangqi") //["zhangqi"]
var newArr = Array.of(true) //[true]
var newArr = Array.of({name: 'zhangqi'}) //[{name: 'zhangqi'}]

4. Array.prototype.concat()

作用：合并两个或多个数组
语法：

  • var newArr = arr1.concat(arr2, arr3, …arrN)
  • var newArr = Array.prototype.call(arr1, arr2, …arrN)

参数：需要与原数组合并的数组或非数组值
返回值：新的 Array 实例
是否改变原数组的值：false
兼容性：IE5.5+
版本：ES3
描述：

let arr1 = ["a", "b", "c"];
let arr2 = ["d", "e", "f"];
let arr3 = ["g"];
let arr4 = arr1.concat(arr2,arr3);
console.log(arr4); // [ "a", "b", "c", "d", "e", "f", "g" ]
console.log(arr1); // ["a", "b", "c"]
console.log(arr2); // ["d", "e", "f"]

5. Array.prototype.copyWithin()

作用：浅复制数组的一部分到同一数组中的另一个位置
语法：

  • arr.copyWithin(target)
  • arr.copyWithin(target, start)
  • arr.copyWithin(target, start, end)
  • arr.copyWithin(目标索引, [源开始索引], [结束源索引])

参数：

  • target: 复制序列到该位置，target < 0 ? target = arr.length + target
  • start: 开始复制的元素，start < 0 ? start = arr.length + start
  • end: 复制元素的结束位置，end < 0 ? end = arr.length + end

返回值：改变了的数组
是否改变原数组的值：true
兼容性：IE-
版本：ES6
描述：类数组不能直接调用copyWithin()方法，要用call/apply调用

var arr = [1,2,3,4,5,6];
arr.copyWithin(2,3); //[1,2,4,5,6,6]
arr.copyWithin(-2); //[1,2,3,4,1,2]
arr.copyWithin(0,3,4); //[4,2,3,4,5,6]
arr.copyWithin(-2,-3,-1); //[1, 2, 3, 4, 4, 5]
[].copyWithin.call({length: 5, 3: 1}, 0, 3); //Object {0: 1, 3: 1, length: 5}

6. Array.prototype.entries()

作用：返回一个新的Array Iterator对象，该对象包含数组中每个索引的键/值对
语法：arr.entries()
参数：无
返回值：一个新的 Array 迭代器对象
是否改变原数组的值：
兼容性：IE-
版本：ES6
描述：—

var arr = ["a", "b", "c"];
var iterator = arr.entries();// ArrayIterator {}

console.log(iterator.next().value); 
// [0, "a"]
console.log(iterator.next().value); 
// [1, "b"]
console.log(iterator.next().value); 
// [2, "c"]

//使用for...of循环
for (let e of iterator) {
    console.log(e);
}
// [0, "a"] 
// [1, "b"] 
// [2, "c"]

7. Array.prototype.every()

作用：测试数组的所有元素是否都通过了指定函数的测试
语法：arr.every(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：false : true(无返回值为false)
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：寻找false,有则返回false,否则返回true

function isOdd(value, index, oldArr){
    return value%2 === 0;
}
var bol = [2,3,4].every(isOdd); //false
var bol = [2,4,6].every(isOdd); //true

8. Array.prototype.fill()

作用：用一个固定值填充一个数组中从起始索引到终止索引内的全部元素
语法：

  • arr.fill(value)
  • arr.fill(value, start)
  • arr.fill(value, start, end)

参数：

  • value: 用来填充数组元素的值
  • start: 起始索引，默认为0 => start < 0 ? start = arr.length + start : start
  • end: 终止索引，默认this.length => end < 0 ? start = arr.length + end : end

返回值：修改后的数组
是否改变原数组的值：true
兼容性：IE-, opera-, safri-
版本：ES6
描述：类数组不能直接调用fill()方法，要用call/apply调用

var arr = [1,2,3,4,5];
arr.fill(6) === arr.fill(6,0) === arr.fill(6,0,arr.length); //[6,6,6,6,6];
arr.fill(6,0,3); //[6,6,6,4,5]
arr.fill(6,2); //[1,2,6,6,6]
[].fill.call({length: 3}, 4); //{0: 4,1: 4,2: 4, length: 3}

9. Array.prototype.filter()

作用：创建一个新数组, 其包含通过所提供函数实现的测试的所有元素
语法：arr.filter(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值： 一个新的通过测试的元素的集合的数组
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：—

var newArr = [1,2,3,4,5,6].filter(function(value, index){
   return value % 2 === 0; 
}); //[2,4,6]

10. Array.prototype.find()

作用：返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。
语法：arr.find(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：数组中满足提供的测试函数的第一个元素的值 || undefined
是否改变原数组的值：false
兼容性：IE12, opera-
版本：ES6
描述：找到符合元素之前，callback函数会为数组中的每个索引调用，而不仅仅是那些被赋值的索引，如果找到了一个这样的元素， 则 find 将会立刻返回这个元素。这意味着对于稀疏数组来说，该方法的效率要低于那些只遍历赋值索引的方法。

var rel = [12, 5, 8, 130, 44].find(function(value, index){
    return value >= 15; 
});
console.log(rel); //130

//寻找数组中的质数
function isPrime(value, index, arr){
    var start = 2;
    if(start <= Math.sqrt(value)){
        if(value%start++ < 1){
            return false;
        }
    }
    return value > 1;
}
console.log([4,6,8].find(isPrime)); //undefined
console.log([4,5,8].find(isPrime)); //5

11. Array.prototype.findIndex()

作用：返回数组中满足提供的测试函数的第一个元素的索引。否则返回-1
语法：arr.findIndex(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：数组中满足提供的测试函数的第一个元素的索引 || -1
是否改变原数组的值：false
兼容性：IE-, opera-
版本：ES6
描述：找到符合元素之前，callback函数会为数组中的每个索引调用，而不仅仅是那些被赋值的索引, 如果找到了一个这样的元素， 则 findIndex 将会立刻返回这个元素的索引。

function isPrime(element, index, array) {
    var start = 2;
    while (start <= Math.sqrt(element)) {
        if (element % start++ < 1) return false;
    }
    return (element > 1);
}

console.log( [4, 6, 8, 12].findIndex(isPrime) ); // -1
console.log( [4, 6, 7, 12].findIndex(isPrime) ); // 2

12. Array.prototype.forEach()

作用：对数组的每个元素执行一次提供的函数
语法：array.forEach(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：undefined
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：遍历操作数组的方法

var newArr = [2,4,6].forEach(function(value, index){
   console.log('a['+ index +']=' + value)
});
a[0]=2
a[1]=4
a[2]=6

13. Array.prototype.includes()

作用：用来判断一个数组是否包含一个指定的值
语法：arr.includes(searchElement, fromIndex)
参数：

  • searchElement: 需要查找的元素值
  • fromIndex: 起始索引，fromIndex < 0 ? arr.length + fromIndex

返回值：true || false
是否改变原数组的值：false
兼容性：IE-
版本：ES6
描述：如果fromIndex 大于等于数组长度 ，则返回 false 。该数组不会被搜索。
如果 fromIndex 为负值，计算出的索引将作为开始搜索searchElement的位置。如果计算出的索引小于 0，则整个数组都会被搜索。

[1, 2, 3].includes(2);     // true
[1, 2, 3].includes(4);     // false
[1, 2, 3].includes(3, 3);  // false
[1, 2, 3].includes(3, -1); // true
[1, 2, NaN].includes(NaN); // true

//fromIndex 大于等于数组长度
var arr = ['a', 'b', 'c'];
arr.includes('c', 3);   //false
arr.includes('c', 100); // false

//计算出的索引小于 0
var arr = ['a', 'b', 'c'];
arr.includes('a', -100); // true
arr.includes('b', -100); // true
arr.includes('c', -100); // true

14. Array.prototype.indexOf()

作用：返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1
语法：

  • arr.indexOf(searchElement)
  • arr.indexOf(searchElement, fromIndex)

参数：

  • searchElement: 需要查找的元素值
  • fromIndex: 起始索引，fromIndex < 0 ? arr.length + fromIndex

返回值：首个被找到的元素在数组中的索引位置; 若没有找到则返回 -1
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：indexOf 使用严格相等（strict equality，即 ===）进行判断 searchElement与数组中包含的元素之间的关系。

var array = [2, 5, 9];
array.indexOf(2);     // 0
array.indexOf(7);     // -1
array.indexOf(9, 2);  // 2
array.indexOf(2, -1); // -1
array.indexOf(2, -3); // 0

//找出指定元素出现的所有位置
var indices = [];
var array = ['a', 'b', 'a', 'c', 'a', 'd'];
var element = 'a';
var idx = array.indexOf(element);
while (idx != -1) {
  indices.push(idx);
  idx = array.indexOf(element, idx + 1);
}
console.log(indices);
// [0, 2, 4]

15. Array.prototype.join()

作用：将数组（或一个类数组对象）的所有元素连接到一个字符串中
语法：arr.join(separator)
参数：

  • separator: 指定一个字符串来分隔数组的每个元素

返回值：一个所有数组元素连接的字符串。如果 arr.length 为0，则返回空字符串(chrome下报错)
是否改变原数组的值：false
兼容性：IE5.5+
版本：ES1
描述：如果元素是undefined 或者null，则会转化成空字符串(chrome下报错)

var b = [2,4,6].join(); //2,4,6
var b = [2,4,6].join(""); //246
var b = [2,4,6].join("#"); //2#4#6

16. Array.prototype.keys()

作用：返回一个新的Array迭代器，它包含数组中每个索引的键
语法：arr.keys()
参数：无
返回值：一个新的 Array 迭代器对象
是否改变原数组的值：false
兼容性：IE-
版本：ES6
描述：—

var a = [2,4,6];
var b = a.keys(); //ArrayIterator {}
console.log(b.next()); //Object {value: 0, done: false}
console.log(b.next()); //Object {value: 1, done: false}
console.log(b.next()); //Object {value: 2, done: false}
console.log(b.next()); //Object {value: undefined, done: true}

//索引迭代器会包含那些没有对应元素的索引
var arr = ["a", ,"c"];
Object.keys(arr); //["0", "2"]
arr.keys(); //[0, 1, 2]

17. Array.prototype.lastIndex()

作用：返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1。从数组的后面向前查找，从 fromIndex 处开始。
语法：arr.lastIndexOf(searchElement, fromIndex = arr.length-1)
参数：

  • searchElement: 需要查找的元素值
  • fromIndex: 起始索引，默认为数组的长度减 1, fromIndex < 0 ? arr.length + fromIndex

返回值：首个被找到的元素在数组中的索引位置 || -1
是否改变原数组的值：
兼容性：IE9+
版本：ES5
描述：lastIndexOf 使用严格相等（strict equality，即 ===）比较 searchElement 和数组中的元素。

var array = [2, 5, 9, 2];
var index = array.lastIndexOf(2); // index is 3
index = array.lastIndexOf(7);     // index is -1
index = array.lastIndexOf(2, 3);  // index is 3
index = array.lastIndexOf(2, 2);  // index is 0
index = array.lastIndexOf(2, -2); // index is 0
index = array.lastIndexOf(2, -1); // index is 3

//查找所有元素
var indices = [];
var array = ['a','b','c','d','a','d'];
var element = 'a';
var idx = arr.lastIndexOf(element);

while(idx !== -1){
    indices.push(idx);
    idx = (idx > 0 ? arr.lastIndexOf(element, idx - 1) : -1);
}
console.log(indices); //[4,2,0];

18. Array.prototype.map()

作用：创建一个新数组，结果是该数组中每一个元素调用提供函数后返回的结果
语法：var newArr = arr.map(callback, thisArg);
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：一个新数组，每个元素都是回调函数的结果
是否改变原数组的值：false || 可以在 callback 执行时改变原数组
兼容性：IE9+
版本：ES5
描述：—

var numbers = [1, 4, 9];
var roots = numbers.map(Math.sqrt); //[1,2,3]

var arr = [1,2,3];
var newArr = arr.map(function(value, index){
    return value * 2;
});//[2,4,6];

var a = [1,2,3];
var newArr = a.map(function(value, index){ //改变原数组
    this[index] = value * 2;
}, a);
console.log(a);  //[2,4,6]

19. Array.prototype.pop()

作用：从数组中删除最后一个元素
语法：arr.pop()
参数：无
返回值：被删除的元素
是否改变原数组的值：true
兼容性：ES3
版本：IE5.5+
描述：如果在一个空数组上调用 pop()，它返回 undefined。

let myFish = ["angel", "clown", "mandarin", "surgeon"];
let popped = myFish.pop();
console.log(myFish); // ["angel", "clown", "mandarin"]
console.log(popped); // surgeon

20. Array.prototype.push()

作用：将一个或多个元素添加到数组的末尾
语法：arr.push(element1, element2, …elementN)
参数：element1,element2, …elementN
返回值：数组的新长度
是否改变原数组的值：true
兼容性：ES3
版本：IE5.5+
描述：—

var numbers = [1, 2, 3];
var len = numbers.push(4);
console.log(numbers); // [1, 2, 3, 4]
console.log(len); // 4

21. Array.prototype.reduce()

作用：对累加器和数组中的每个元素(从左到右)应用一个函数，将其减少为单个值
语法：arr.reduce(callback, initialValue)
参数：

  • callback(value, index, oldArr): 测试函数
    • accumulator: 上一次调用回调返回的值，或者是initialValue
    • currentValue: 正在处理的元素值
    • index: 元素索引
    • oldArr: 原数组
  • initialValue: 可选，用于第一次调用callback的初始值(第一个参数)

返回值：函数累计处理的结果
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：回调函数第一次执行时，accumulator 和 currentValue 的取值有两种情况：调用 reduce 时提供initialValue，accumulator 取值为 initialValue ，currentValue 取数组中的第一个值，索引从0开始；没有提供 initialValue ，accumulator 取数组中的第一个值，currentValue 取数组中的第二个值，索引从1开始。

var arr = [1,2,3,4,5];
var num = arr.reduce(function(sum, val, i, arr){
    return sum + val;
},0);
console.log(num); //15

//数组扁平化
var flattened = [[0, 1], [2, 3], [4, 5]].reduce(function(a, b) {
    return a.concat(b);
}, []);
// flattened is [0, 1, 2, 3, 4, 5]

//计算数组中各个值出现次数
var names = ['Alice', 'Bob', 'Tiff', 'Bruce', 'Alice'];
var countedNames = names.reduce(function(allNames, name) { 
  if (name in allNames) {
    allNames[name]++;
  }
  else {
    allNames[name] = 1;
  }
  return allNames;
}, {});
// countedNames is { 'Alice': 2, 'Bob': 1, 'Tiff': 1, 'Bruce': 1 }

22. Array.prototype.reduceRight()

作用：对累加器和数组中的每个元素(从右到左)应用一个函数，将其减少为单个值
语法：同reduce
参数：同reduce
返回值：函数累计处理的结果
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：首次调用回调函数时，previousValue 和 currentValue 可以是两个值之一。如果调用 reduceRight 时提供了 initialValue 参数，则 previousValue 等于 initialValue，currentValue 等于数组中的最后一个值。如果没有提供 initialValue 参数，则 previousValue 等于数组最后一个值， currentValue 等于数组中倒数第二个值。

reduce和reduceRight计算方向不同    
var a = ['1', '2', '3', '4', '5']; 
var left  = a.reduce(function(prev, cur)      { return prev + cur; }); 
var right = a.reduceRight(function(prev, cur) { return prev + cur; }); 
console.log(left);  // "12345"
console.log(right); // "54321"

23. Array.prototype.reverse()

作用：将数组中元素的位置颠倒
语法：arr.reverse()
参数：无
返回值：该数组的引用
是否改变原数组的值：true
兼容性：IE5.5+
版本：ES1
描述：reverse 方法颠倒数组中元素的位置，并返回该数组的引用。

var arr = [1,2,3,4,5];
var b = arr.reverse();
console.log(arr); //[5,4,3,2,1]
console.log(b);   //[5,4,3,2,1]

语法：arr.shift()
参数：无
返回值：被删除的元素 || 数组为空，返回undefined
是否改变原数组的值：true
兼容性：IE5.5+
版本：ES3
描述：—

let a = [1, 2, 3];
let b = a.shift();
console.log(a); // [2, 3]
console.log(b); // 1

25. Array.prototype.slice()

作用：返回一个从开始到结束（不包括结束）选择的数组的一部分浅拷贝到一个新数组对象
语法：

  • arr.slice() //默认从0开始
  • arr.slice(begin) //end被省略，提取到数组末尾
  • arr.slice(begin, end) //end大于数组长度，同上

参数：

  • begin: 起始索引
  • end: 结束索引
  • 如果索引为负数: begin = arr.length + begin | end = arr.length + end

返回值：含有提取元素的新数组
是否改变原数组的值：false
兼容性：IE+
版本：ES3
描述：

  • 如果该元素是对象引用，如果发生改变，新旧数组中的元素都会发生改变

  • 对于字符串，数字，布尔值，新数组修改不会影响原数组

    var arr = [1,2,3,[4,5]];
    var b = arr.slice();
    b[3].push(6)
    console.log(arr); //[1,2,3,[4,5,6]]
    console.log(b); //[1,2,3,[4,5,6]]

    var arr = [1,2,3,[4,5]];
    var b = arr.slice();
    b.push(6);
    console.log(arr); //[1,2,3,[4,5]]
    console.log(b); //[1,2,3,[4,5],6]

26. Array.prototype.some()

作用：测试数组中的某些元素是否通过由提供的函数实现的测试
语法：arr.some(callback, thisArg)
参数：

  • callback(value, index, oldArr): 测试函数
    • value: 每一项的元素值
    • index: 元素索引
    • oldArr: 原数组
  • thisArg: 执行callback时的this值

返回值：true : false
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：寻找true,有则返回true,否则返回false

function isOdd(value, index, oldArr){
    return value%2 === 0;
}
var bol = [2,3,5].every(isOdd); //true
var bol = [3,5,11].every(isOdd); //false

27. Array.prototype.sort()

作用：在适当的位置对数组的元素进行排序，并返回数组
语法：arr.sort() || arr.sort(compareFunction)
参数：compareFunction: 可选, 用来指定按某种顺序进行排列的函数
返回值：排序后的数组
是否改变原数组的值：true
兼容性：IE5.5+
版本：ES1
描述：sort 排序不一定是稳定的。默认排序顺序是根据字符串Unicode码点。
如果 compareFunction(a, b) 小于 0 ，那么 a 会被排列到 b 之前；
如果 compareFunction(a, b) 等于 0 ， a 和 b 的相对位置不变。备注： ECMAScript 标准并不保证这一行为，而且也不是所有浏览器都会遵守
如果 compareFunction(a, b) 大于 0 ， b 会被排列到 a 之前。
compareFunction(a, b) 必须总是对相同的输入返回相同的比较结果，否则排序的结果将是不确定的。

function compareNumbers(a, b){//倒序排列
    return b - a;
}
var arr = [4,1,3,6,2];
arr.sort(compareNumbers); //[6,4,3,2,1]

//对非 ASCII 字符排序
//一些非英语语言的字符串需要使用String.localeCompare。这个函数可以将函数排序到正确的顺序。
var items = ['réservé', 'premier', 'cliché', 'communiqué', 'café', 'adieu'];
items.sort(function (a, b) {
  return a.localeCompare(b);
});
// items is ['adieu', 'café', 'cliché', 'communiqué', 'premier', 'réservé']

//使用映射改善排序
var list = ['Delta', 'alpha', 'CHARLIE', 'bravo'];
// 对需要排序的数字和位置的临时存储
var mapped = list.map(function(el, i){
    return { index: i, value: el.toLowerCase() };
});
// 按照多个值排序数组
mapped.sort(function(a, b){
    return +(a.value > b.value);
});
// 根据索引得到排序的结果
var result = mapped.map(function(el){
    return list[el.index];
});
console.log(result); //["alpha", "bravo", "CHARLIE", "Delta"]

28. Array.prototype.splice()

作用：通过删除现有元素和/或添加新元素来更改一个数组的内容
语法：

  • arr.splice(start) //删除start(包括start)后所有元素
  • arr.splice(start, deleteCount)
  • arr.splice(start, deleteCount, item1, item2, …)

参数：

  • start: 指定修改的开始位置，start > arr.length ? start = arr.length, start < 0 ? start = start + arr.length
  • deleteCount: 要移除元素的个数, 0: 不移除。如果 deleteCount 大于start 之后的元素的总数，则从 start 后面的元素都将被删除（含第 start 位）。
  • item1,item2,…: 要添加进数组的元素,从start 位置开始。如果不指定，则 splice() 将只删除数组元素。

返回值：由被删除的元素组成的一个数组。如果只删除了一个元素，则返回只包含一个元素的数组。如果没有删除元素，则返回空数组。
是否改变原数组的值：true
兼容性：ie5.5+
版本：ES3
描述：如果添加进数组的元素个数不等于被删除的元素个数，数组的长度会发生相应的改变。

var arr = [1,2,3,4,5];
arr.splice(1); //[1]  返回值: [2,3,4,5]
arr.splice(1,2,7,8,9); //[1, 7, 8, 9, 4, 5]  返回值: [2,3]

31. Array.prototype.toString()

作用：返回一个字符串，表示指定的数组及其元素。
语法：arr.toString()
参数：无
返回值：字符串
是否改变原数组的值：false
兼容性：all
版本：ES1
描述：

  • Array 对象覆盖了 Object 的 toString 方法。对于数组对象，toString 方法返回一个字符串，该字符串由数组中的每个元素的 toString() 返回值经调用 join() 方法连接（由逗号隔开）组成。

  • toString() 方法是通用的，可被用于任何对象。如果对象有一个 join() 方法，将会被调用，其返回值将被返回。没有则调用 Object.prototype.toString() 方法。

    var monthNames = [‘Jan’, ‘Feb’, ‘Mar’, ‘Apr’];
    var myVar = monthNames.toString(); // assigns “Jan,Feb,Mar,Apr” to myVar.
