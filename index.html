Array对象内置方法总结

发表于 2017-06-23

1. Array.from()

作用：从一个类似数组或可迭代的对象中创建一个新的数组实例
语法：Array.from(arraylike, mapFn, thisArg);
参数：

  • arraylike: 类数组对象或可遍历对象(有length属性的对象)
  • mapFn: 可选参数，用来处理数组的函数，返回新数组
  • thisArg: 可选参数，执行mapFn时的this值

返回值：新的Array实例
是否改变原数组的值：false
兼容性：IE不支持
版本：ES5
描述：Array.from(obj, mapFn, thisArg) 类似于 Array.from(obj).map(mapFn, thisArg)

const bar = ['a','b','c'];
var newArr = Array.from(bar);//['a','b','c']

//等同于string.split("");
var newArr = Array.from('foo') //['f','o','o'];

var newArr = Array.from({length: 5},function(val,i){
    return i;
}); //[0,1,2,3,4];

2. Array.isArray()

作用：用于确定传递的值是否是一个 Array
语法：Array.isArray(obj)
参数：

  • obj: 需要检测的值

返回值：array ? true : false
是否改变原数组的值：false
兼容性：IE9+
版本：ES5
描述：—

Array.isArray([1, 2, 3]);  
// true
Array.isArray({foo: 123}); 
// false

3. Array.of()

作用：创建一个具有可变数量参数的新数组实例,而不考虑参数的数量或类型
语法：Array.of(elem0, elem1, elem2, …elemN)
参数：任意个参数，将按顺序成为返回数组中的元素
返回值：新的 Array 实例
是否改变原数组的值：false
兼容性：IE,Opera不支持
版本：ES6
描述：—

var newArr = Array.of(1) //[1]
var newArr = Array.of(1,2,3) //[1,2,3]
var newArr = Array.of("zhangqi") //["zhangqi"]
var newArr = Array.of(true) //[true]
var newArr = Array.of({name: 'zhangqi'}) //[{name: 'zhangqi'}]

4. Array.prototype.concat()

作用：合并两个或多个数组
语法：

  • var newArr = arr1.concat(arr2, arr3, …arrN)

  • var newArr = Array.prototype.call(arr1, arr2, …arrN)
    参数：需要与原数组合并的数组或非数组值
    返回值：新的 Array 实例
    是否改变原数组的值：false
    兼容性：IE5.5+
    版本：ES3
    描述：

    let arr1 = [“a”, “b”, “c”];
    let arr2 = [“d”, “e”, “f”];
    let arr3 = [“g”];
    let arr4 = arr1.concat(arr2,arr3);
    console.log(arr4); // [ “a”, “b”, “c”, “d”, “e”, “f”, “g” ]
    console.log(arr1); // [“a”, “b”, “c”]
    console.log(arr2); // [“d”, “e”, “f”]

Boolean对象内置方法总结

发表于 2017-06-23

1. toString

参数： 无
作用： 返回指定的布尔对象的字符串形式
返回值：指定的布尔对象的字符串形式
描述： 当一个布尔对象作为文本值或进行字符串连接时，JavaScript 会自动调用其toString 方法

var flag = true;
var myVar = flag.toString(); // "true"
console.log(typeof a); //string

2. valueOf

参数：无
作用：返回一个Boolean对象的原始值
返回值：Boolean对象的原始值
描述：该方法通常在 JavaScript 内部调用，而不是在代码中显示调用

var bol = true;
var a = bol.valueOf(); //true
console.log(typeof a); //boolean

3. toSource(不是任何标准的一部分)

参数：无
作用：返回一个表示对象的源码的字符串
返回值：返回一个表示对象的源码的字符串
